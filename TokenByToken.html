Token by Token

My notes on machine learning, as I learn it.

These are the notes I wish I had when I started. Written to understand things properly, not to impress anyone — open to whoever finds them useful.

I kept running into the same problem: most explanations were either too hand-wavy to be useful, or assumed I already knew the thing I was trying to learn.

So I started writing my own notes. The rule I set for myself: don't move on until I can explain it clearly. If I can't write it out without leaning on jargon, I don't actually understand it yet.

These aren't polished tutorials. They're how I actually learned this stuff — and they're here in case they help you too.

How to use these notes

  • Each series follows the order I learned things in — start from the top
  • The math is there when it matters, skipped when it doesn't add anything
  • If something's unclear, that's a gap in my notes — I'm still filling them in
